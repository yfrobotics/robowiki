# RoboWiki --- 机器人维基百科

(project logo)

## 介绍

本维基旨在包含机器人设计、编程、理论、算法、模型、资源等方面的内容。

我们采用公共知识编辑的方式提倡**知识自由**。



## 编译与部署

`sudo python3 -m pip install -r "requirements.txt"`

`mkdocs --version`

`mkdocs build`

或者 

`mkdocs serve`

打开浏览器，访问：`localhost:8000`
